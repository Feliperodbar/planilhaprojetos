# Gestão de Atividades - React + TypeScript + Vite

Projeto com controle de tarefas de projetos, incluindo:

- Cadastro, edição e exclusão de atividades
- Persistência em LocalStorage
- Filtros por status e responsável
- Ordenação por colunas
- Paginação simples
- Exportação para CSV
- Destaque visual por status e atraso

## Como rodar

1. Instale dependências:

```bash
npm install
```

2. Execute em desenvolvimento:

```bash
npm run dev
```

3. Build de produção:

```bash
npm run build
```
